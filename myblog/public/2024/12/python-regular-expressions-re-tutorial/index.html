Python Regular Expressions (re) Tutorial

Regular expressions (re) in Python are a powerful tool for text processing. This tutorial will guide you through the basics and practical usage of re.

Importing the re Module

1

import re

Key Functions in re

1. re.match(pattern, string)

Checks if the pattern matches at the beginning of the string.

1
2
3

result = re.match(r"Hello", "Hello, World!")
if result:
    print(result.group())  # Output: Hello

2. re.search(pattern, string)

Searches for the first occurrence of the pattern in the string.

1
2
3

result = re.search(r"World", "Hello, World!")
if result:
    print(result.group())  # Output: World

3. re.findall(pattern, string)

Returns a list of all occurrences of the pattern.

1
2

result = re.findall(r"\d+", "The numbers are 123 and 456.")
print(result)  # Output: ['123', '456']

4. re.sub(pattern, replacement, string)

Replaces occurrences of the pattern with a replacement.

1
2

result = re.sub(r"\s+", "-", "Hello World Python")
print(result)  # Output: Hello-World-Python

5. re.compile(pattern)

Compiles a pattern for reuse.

1
2
3

pattern = re.compile(r"\d+")
result = pattern.findall("Numbers: 123, 456, 789")
print(result)  # Output: ['123', '456', '789']

Common Regex Patterns

Pattern Description Example
\d Matches a digit 0, 1, 2, 3
\w Matches word characters a, b, 1, _
\s Matches whitespace (space), \t
. Matches any character a, 1, @
^ Matches the start of a string ^Hello
$ Matches the end of a string World$

Examples

1. Validating an Email Address

1
2
3
4
5
6

email = "example@example.com"
pattern = r"^\w+@\w+\.\w+$"
if re.match(pattern, email):
    print("Valid Email")
else:
    print("Invalid Email")

2. Extracting Dates

1
2
3
4

text = "Today is 2024-12-08. Tomorrow is 2024-12-09."
pattern = r"\d{4}-\d{2}-\d{2}"
dates = re.findall(pattern, text)
print(dates)  # Output: ['2024-12-08', '2024-12-09']

3. Removing Extra Spaces

1
2
3

text = "This   is   a    test."
result = re.sub(r"\s+", " ", text)
print(result)  # Output: This is a test.
